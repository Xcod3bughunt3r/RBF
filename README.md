# Redis Brute Force

![!](/rbf.png)

- <kbd>Download / Install</kbd>

```cmd
root@kali:~# wget -q "https://raw.githubusercontent.com/Xcod3bughunt3r/RBF/main/RBF.sh" -O /usr/bin/rbf
root@kali:~# chmod +x /usr/bin/rbf
```

- <kbd>Use</kbd>

```cmd
root@kali:~# rbf -t <TARGET> -p <PORT> -w <WORDLIST>
```
